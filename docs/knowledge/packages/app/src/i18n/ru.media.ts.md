# packages/app/src/i18n/ru.media.ts

Русский i18n-фрагмент зоны `media` — пара к `en.media.ts`, типизирована как `typeof mediaEn` (расхождение форм — ошибка tsc, не рантайма). С Э0 веб-паритета (2026-07-15) живёт в `@muza/app`. Обновлено: 2026-07-15 (переезд).

---

Дата: 2026-07-14 (заполнение), 2026-07-15 (переезд в пакет).

Ключи и структура 1:1 с `en.media.ts` (см.
`docs/knowledge/packages/app/src/i18n/en.media.ts.md` — там расписаны все
зоны). Значения — оригинальные русские строки, ранее захардкоженные прямо в
`player/*`/`lib/*` (T-media, эпик W5).

2026-07-16: из `statsBlocks.*` удалён ключ `wrapped` — синхронно с
`en.media.ts` (блок «Итоги года» убран со страницы статистики).

Нетривиальное: `search.versions` хранит русское склонение
(`one`/`few`/`many` = «версия»/«версии»/«версий»), EN-сторона для `few`/`many`
дублирует «versions» (в английском только 2 формы) — намеренная асимметрия
значений при одинаковой форме ключей.
