# apps/desktop/src/shell/ImportDialog.tsx

«Импорт плейлиста» (Stage 4): ссылка на YT/YTM/Spotify/Apple Music → сервер
матчит позиции в каталог и создаёт плейлист. Показывает честный отчёт
(сколько нашли / что не нашлось), без иллюзии стопроцентного переноса.

---

**i18n (2026-07-14, T34a, эпик W5):** строки извлечены в
`dialogs.importPlaylist.*` через `const { t } = useT();`. Кнопка «Отмена»
переиспользует `common.cancel` (идентичный текст, не заводился отдельный
ключ). Данные отчёта (`u.artist — u.title` в списке ненайденных) — не
переводимый текст, просто интерполяция данных, оставлено как есть.
