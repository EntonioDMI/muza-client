# apps/desktop/src/plugins/PluginFrames.tsx

Фреймы плагинов уровня 1 (эпик W8, T44): по одному живому `<iframe>` на
включённый плагин. Фрейм никогда не меняет родителя (иначе перезагрузка
realm) — видимость/геометрия задаётся только CSS по активной поверхности
(скрыт-фон / вкладка / панель / оверлей). `sandbox="allow-scripts"` без
`allow-same-origin` → opaque origin.

---

Два компонента: `PluginFrame` (один `<iframe>`) и `PluginFrames` (список +
кнопки закрытия активной поверхности). Рендерится из `App.tsx` внутри
`<LanguageProvider>`.

**i18n (2026-07-14, T34a, эпик W5, общий свип):** три label кнопок закрытия
поверхности (оверлей/панель/вкладка плагина) были захардкожены по-русски —
найдены отдельным свипом ПОСЛЕ основных 10 shell-диалогов (не входили в
исходный список T34a, но той же природы: EN-дефолт пользователь видел бы
русский). Извлечены в новую зону `plugins.{closeOverlay,closePanel,closeTab}`
через `const { t } = useT();`.
