# Changelog

Формат: версии сверху вниз, в каждой — «Добавлено / Изменено / Починено».
Готовится под парсинг Discord-ботом релизов (см. память проекта: сводка релиза обязательна).

## 0.1.4 — 2026-07-16

### Добавлено
- Настраиваемая длительность кроссфейда (ползунок 1–12 с в «Воспроизведении»).
- Согласие на анонимную статистику при регистрации: явная галочка + «Подробнее» (что именно отправляется).
- Админ-панель: вкладки «Рост» и «Ошибки» — детали ошибок по клику, очистка, фильтры по типу/версии.
- Нотка-финал под текстом песни — декоративный знак внизу лирики (настраивается, по умолчанию включена).
- Иконка плейлиста из обложки трека: ПКМ по треку в плейлисте → «Сменить иконку».
- «Любимое» закреплено первой плиткой в Библиотеке и первой строкой в сайдбаре (как в Spotify).

### Изменено
- Карточка «Итоги года» — минималистичный редизайн: меньше текста, больше графики.
- Единые красивые подсказки у всех кнопок — стандартные браузерные (с белой обводкой) убраны.
- Discord Rich Presence: обложка трека показывается без чёрных рамок.
- Навигатор настроек: анимированная «пилюля» при переключении разделов вместо fade; выровнены отступы.
- Ползунок масштаба интерфейса больше не дёргает экран при перетаскивании.

### Починено
- Повтор трека и повтор плейлиста больше не останавливаются сами по себе.
- Радио не встаёт из-за одного нерабочего трека — молча пропускает и играет дальше.
- Множество SoundCloud-треков снова играют (исправлены битые ссылки источников в каталоге).
- Полоска прогресса не показывает минуту прошлого трека при переходе на следующий.
- Настройки и вход в аккаунт переживают «Завершить задачу» / жёсткое закрытие приложения.
- Одиночный клик по строке текста перематывает на неё; аннотация Genius — двойным кликом.
- Приём телеметрии защищён от накрутки, старые ошибки авто-очищаются.
