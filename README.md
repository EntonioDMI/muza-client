# Muza

Открытый десктоп-клиент музыкального плеера. Тексты-герой (синхро-караоке, без цензуры и блюра), тотальная кастомизация, лёгкий клиент на Tauri.

> Muza не хостит и не распространяет аудио. Клиент воспроизводит контент из публичных источников на устройстве пользователя. Не аффилирован ни с одним музыкальным сервисом.

## Структура

Монорепо pnpm + Turborepo:

- `apps/desktop` — десктоп-приложение (Tauri v2 + Vite + React + TS)
- `packages/ui` — дизайн-система Muza (токены + компоненты)
- `packages/core` — доменная логика плеера (очередь, воспроизведение)
- `packages/api-client` — типизированный контракт API (Zod)

## Разработка

```bash
pnpm install
pnpm desktop        # запуск десктоп-приложения (tauri dev)
```

Требования: Node 22+, pnpm 10+, Rust stable (MSVC), WebView2.
